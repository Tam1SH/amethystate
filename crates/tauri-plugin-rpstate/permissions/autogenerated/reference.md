## Default Permission

Allows access to the default rpstate commands

#### This default permission set includes the following:

- `allow-rpstate-get`
- `allow-rpstate-set`
- `allow-rpstate-subscribe`
- `allow-rpstate-unsubscribe`
- `allow-rpstate-get-prefix`
- `allow-rpstate-flush`
